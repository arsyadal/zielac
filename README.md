A premium, high-converting landing page for **SERVICE AC KARAWANG Ziel AC**. Designed to showcase professional HVAC services with a modern, responsive interface.

## 🚀 Features

- **Professional Design**: Modern aesthetic using Navy and Cyan color palettes.
- **24/7 Service Info**: Prominent display of round-the-clock availability.
- **WhatsApp Integration**: Floating action button and Hero CTAs for instant communication.
- **Interactive Maps**: Integrated Google Maps for easy location finding.
- **Customer Reviews**: Showcasing high satisfaction with a 5.0 Google rating.
- **Mobile Responsive**: Fully optimized for all screen sizes.

## 🛠️ Tech Stack

- **Frontend**: React.js
- **Build Tool**: Vite
- **Styling**: Vanilla CSS with custom design tokens
- **Assets**: Professional AI-generated HVAC imagery

## 📦 Installation & Setup

1. **Clone the repository**
2. **Install dependencies**:
   ```bash
   npm install
   ```
3. **Run the development server**:
   ```bash
   npm run dev
   ```
4. **Build for production**:
   ```bash
   npm run build
   ```
